# Modelo de Caixas Cap. 16

## Anatomia de uma Caixa
- Height == Altura
- Widht == Largura
- Border == Borda (Linha que circula o conteúdo)
- Padding == Preenchimento (Espaço entre o texto e a border // Espaço interno)
- Margin == Margem (Espaço Externo)
- Outline == Contorno

## Tipos de Caixas
- Box-Level
    - Pula Para a linha de baixo
    - Sempre se inicia em uma linha nova
    - Ocupa a largura total da tela
    - EX: &lt;div&gt;, &lt;h1&gt;, &lt;p&gt;, &lt;main&gt;

- inline-level
    - Não pula para próxima linha
    - Não ocupa a largura inteira
    - E não quebra a linha
    - EX: &lt;span&gt;, &lt;a&gt;, &lt;strong&gt;
