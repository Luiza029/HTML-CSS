Entanda mais sobre a linguagem

JavaScript (JS) é uma linguagem de programação leve, interpretada e baseada em protótipos, usada principalmente para tornar páginas da web interativas. Juntamente com HTML (estrutura) e CSS (estilo), é uma das três principais tecnologias da World Wide Web.

Criado em 1995 por Brendan Eich na Netscape Communications, originalmente como Mocha, depois renomeado para LiveScript e finalmente JavaScript (com influência de Java, apesar de ser uma linguagem distinta).

Curiosidades sobre a linguagem

  • NaN (Not-a-Number) é um valor especial que representa resultados matemáticos inválidos, e NaN não é igual a si mesmo, ou seja, NaN !== NaN é verdadeiro.
  • Com o surgimento do Node.js, JavaScript deixou de ser apenas uma linguagem client-side e passou a ser usada no backend, tornando possível o desenvolvimento full-stack com uma única linguagem.
  • A linguagem é interpretada diretamente pelo navegador, sem necessidade de compilação, permitindo visualização imediata dos resultados e acelerando o desenvolvimento.
